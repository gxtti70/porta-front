Interfaz de usuario de alto rendimiento diseñada para presentar mi trayectoria como **Junior Full Stack Developer**. Enfocada en la velocidad, el diseño responsivo y la interactividad.

## 🌟 Valor Diferencial
* **Diseño Responsive:** Optimizado para dispositivos móviles y escritorio bajo estándares modernos.
* **Componentes de Alto Nivel:** Desarrollados con **Tailwind CSS** para un look "Dark Mode" elegante y minimalista.
* **Chatbot Interactivo:** Integración directa con el backend para ofrecer respuestas en tiempo real sobre mis habilidades.
* **Clean Code:** Arquitectura basada en componentes reutilizables y tipado estricto con **TypeScript**.

## 🛠️ Tech Stack
* **Frontend:** React 18 + Vite.
* **Estilos:** Tailwind CSS + Framer Motion (para animaciones).
* **Iconografía:** React Icons (Font Awesome & Simple Icons).
* **Despliegue:** Vercel con CI/CD automatizado.

## 🚀 Configuración
1. Clonar el repositorio.
2. Instalar dependencias: `npm install`.
3. Configurar variable de entorno `.env`: `VITE_API_URL=https://porta-back.onrender.com`.
4. Ejecutar desarrollo: `npm run dev`.
